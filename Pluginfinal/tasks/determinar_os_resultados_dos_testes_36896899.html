Task: Determinar os resultados dos testes
Purpose
 
Necessário para averiguar se os testes realizados tiveram o resultado correto e esperado em cada função.
Relationships
RolesPrimary Performer:
  • analista_de_teste
  • testador
Additional Performers:
InputsMandatory:
  • Diretrizes de teste (Catálogo de ideias de teste)
  • Plano de desenvolvimento de software
  • Plano de testes
    		Optional:
    • None
    Outputs
    • Diretrizes de teste (Catálogo de ideias de teste)
    • Plano de testes
      Steps
      Examinar todos os incidentes e falhas de teste
       
      Necessário para determinar onde se encontram incidentes e falhas
      Criar e manter solicitações de alteração
       
      Deve ser mantida uma lista de alterações a serem feitas para corrigir as falhas
      Analisar e avaliar o status
       
      Deve ser realizada a checagem do status atual dos detalhes solicitados
      Aconselhar as partes interessadas sobre as principais conclusões
       
      Uma forma de feedback deve ser realizada para as partes interessadas para indicar o status dos aspectos testados
      Avaliar e verificar seus resultados
       
      Necessário para checar se houve correção das falhas
