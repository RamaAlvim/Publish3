Task: Identificar ideias de teste
Purpose
 
Função básica para identificar os testes necessários e para sua realização.
Relationships
RolesPrimary Performer:
  • analista_de_teste
  • testador
Additional Performers:
InputsMandatory:
  • Diretrizes de teste (Catálogo de ideias de teste)
  • Plano de desenvolvimento de software
  • Plano de testes
    		Optional:
    • None
    Outputs
    • Diretrizes de teste (Catálogo de ideias de teste)
    • Plano de testes
      Steps
      Identificar motivadores de teste e itens de teste de alvos relevantes
       
      Função básica para realizar a tarefa de teste
      Examinar os catálogos relevantes de ideias de testes disponíveis
       
      Esse processo acelera a listagem de ideias de teste relevantes
      Brainstorm de ideias de testes adicionais
       
      Necessário para criar uma lista de ideias de teste
      Listar candidados a ideia de teste
       
      Necessário para que sejam excluidas as ideias de teste inválidas
      Redefinir a lista de ideias de teste
       
      As novas ideias de teste devem ser adicionadas à lista
      Avaliar e verificar os resultados
       
      Fundamental para se chegar a uma lista de ideias de teste final
